📖 Dokumentasi

Panduan Lengkap

Dokumentasi lengkap untuk instalasi, konfigurasi, dan penggunaan Financier.

🚀 Memulai

Persyaratan

  • Flutter SDK >=3.7.0
  • Dart SDK >=3.7.0
  • Akun Supabase (gratis)
  • Android Studio / VS Code (opsional)

Clone & Install

git clone https://github.com/Ferdi-89/economic.git
cd economic/Economic/financier
flutter pub get

💾 Setup Supabase

  1. Buat project baru di supabase.com
  2. Buka SQL Editor dan paste isi supabase/migrations/00001_initial_schema.sql
  3. Jalankan query untuk membuat tabel, RLS policy, trigger, dan seed data
  4. Ambil URL dan anon key dari Settings → API

Environment Variables

Salin file .env.example menjadi .env dan isi kredensial:

SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key

Atau gunakan --dart-define saat build:

flutter run --dart-define=SUPABASE_URL=https://xxx.supabase.co \
  --dart-define=SUPABASE_ANON_KEY=xxx

🏠 Struktur Proyek

financier/
  lib/
    config/           # Konfigurasi app, env, Supabase
    core/             # Theme, extensions, utils
    data/
      models/         # Freezed models (Account, Transaction, dll)
      repositories/   # Auth, Transaction, Account, Category, Budget
      datasources/    # Hive local storage
    domain/
      entities/       # Domain objects
      usecases/       # Business logic
    presentation/
      router/         # GoRouter config
      screens/        # Dashboard, Transaksi, Akun, Budget, Laporan, Settings
      widgets/        # Shared widgets

🎨 Build Aplikasi

Android APK

flutter build apk --release

Hasil build ada di build/app/outputs/flutter-apk/app-release.apk

Web (PWA)

flutter build web
flutter run -d chrome

Debug Mode

flutter run
flutter run -d chrome  # Web

💡 Fitur Utama

Multi Rekening

  • 5 jenis: Cash, Bank, E-Wallet, Savings, Investment
  • Setiap rekening memiliki saldo terpisah
  • Rekening bisa diarsipkan (tanpa hapus histori)
  • Lihat total saldo di halaman daftar rekening

Kategori

  • 16 kategori bawaan (makanan, transportasi, belanja, dll)
  • Buat kategori sendiri
  • Kategori terpisah untuk pemasukan dan pengeluaran
  • Urutan bisa diatur dengan sort order

Transaksi

  • 3 tipe: Pemasukan, Pengeluaran, Transfer
  • Pilih rekening sumber dan kategori
  • Catatan opsional untuk setiap transaksi
  • Riwayat transfer tercatat otomatis di 2 rekening
  • Filter berdasarkan tanggal, kategori, atau rekening
  • Edit dan hapus transaksi yang sudah dibuat

Budget

  • Atur budget per kategori
  • Progress bar visual
  • Peringatan saat 80% terpakai
  • Over-budget alert (merah)
  • Periode: bulanan, mingguan, tahunan, custom

Laporan

  • Grafik batang 6 bulan (pemasukan vs pengeluaran)
  • Breakdown pengeluaran per kategori
  • Ringkasan dashboard bulanan

☁️ Sinkronasi & Offline

  • Online: Data langsung tersimpan di Supabase
  • Offline: Data disimpan di Hive (local DB)
  • Sync: Saat koneksi tersedia, data offline otomatis diunggah
  • Conflict: Menggunakan last-write-wins

📱 PWA (Progressive Web App)

  1. Build web: flutter build web
  2. Deploy folder build/web ke Vercel/Netlify
  3. Buka URL di browser → install ke home screen
  4. Aplikasi berjalan mandiri tanpa browser chrome
