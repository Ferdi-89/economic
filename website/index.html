Fitur Unggulan

Semua yang Anda butuhkan mengatur keuangan

Dari catatan transaksi harian hingga laporan keuangan bulanan, Financier hadir dengan fitur lengkap untuk kebutuhan finansial Anda.

💳

Multi Rekening

Cash, Bank, E-Wallet, Tabungan, Investasi — kelola semua saldo dalam satu tempat dengan tampilan terpisah.

📈

Pemasukan & Pengeluaran

Catat setiap transaksi dengan kategori yang lengkap. Filter berdasarkan tanggal, kategori, atau rekening.

📊

Budget Bulanan

Atur anggaran per kategori dengan progress bar visual. Dapatkan notifikasi saat mendekati batas.

📑

Laporan Grafik

Analisis tren keuangan 6 bulan terakhir dengan bar chart. Lihat breakdown pengeluaran per kategori.

☁️

Sinkronasi Cloud

Data tersimpan aman di Supabase PostgreSQL. Akses dari perangkat mana pun, kapan pun secara realtime.

🖱️

Offline-first

Catat transaksi tanpa koneksi internet. Data disimpan lokal dan akan sinkron otomatis saat online.

🔄

Transfer Rekening

Pindahkan uang antar rekening dengan sekali ketuk. Riwayat transfer tercatat otomatis di kedua rekening.

🎨

Dark Mode

Nyaman di mata siang maupun malam. Otomatis mengikuti tema sistem atau bisa diatur manual.

🏠

Dashboard Ringkasan

Lihat semua informasi penting di satu layar: saldo, transaksi terbaru, dan ringkasan bulanan.

Tech Stack

Dibangun dengan teknologi modern

Kombinasi teknologi terbaik untuk performa maksimal di Android dan Web.

Fl

Flutter

UI framework cross-platform dari Google. Material 3, animasi halus, performa native.

Sb

Supabase

Backend-as-a-Service. Auth, PostgreSQL, dan Realtime sync tanpa backend server.

Rv

Riverpod

State management deklaratif. Type-safe, testable, dan efisien untuk aplikasi skala besar.

Hv

Hive

Database lokal super cepat. Mendukung offline-first dengan antrian sinkronasi.

Fc

FL Chart

Library grafik interaktif untuk visualisasi data keuangan yang informatif dan menarik.

Gr

GoRouter

Navigasi deklaratif dengan auth guard dan bottom navigation shell untuk UX mulus.
