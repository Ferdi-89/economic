Cara Instalasi

🤖

Android APK

  1. Download file APK dari tautan di atas
  2. Buka file APK di perangkat Android Anda
  3. Izinkan instalasi dari sumber tidak dikenal
  4. Selesai! Financier siap digunakan
🌐

Web PWA

  1. Buka URL Financier di browser (Chrome/Edge/Safari)
  2. Klik "Install" atau "Add to Home Screen"
  3. Aplikasi akan muncul di layar beranda
  4. Akses kapan saja, offline support
