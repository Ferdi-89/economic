🛡️ Privasi

Kebijakan Privasi

Financier menghormati privasi Anda. Berikut bagaimana data Anda dikelola.

Terakhir diperbarui: Juli 2026

Data yang Dikumpulkan

Financier mengumpulkan data minimal yang diperlukan untuk berfungsi:

  • Akun: Alamat email dan nama (untuk autentikasi)
  • Data Keuangan: Transaksi, saldo rekening, kategori, budget yang Anda masukkan secara manual
  • Preferensi: Tema (terang/gelap/sistem) yang disimpan di perangkat lokal

Data yang Tidak Dikumpulkan

  • ✔️ Tidak ada data lokasi
  • ✔️ Tidak ada data kontak
  • ✔️ Tidak ada data biometrik
  • ✔️ Tidak ada pelacakan iklan
  • ✔️ Tidak ada analytics pihak ketiga
  • ✔️ Tidak ada cookie

Penyimpanan Data

Data Anda disimpan di dua tempat:

  • Lokal (Perangkat Anda): Database Hive di perangkat Android atau IndexedDB di browser. Data tetap bisa diakses tanpa koneksi internet.
  • Cloud (Supabase): Database PostgreSQL dihosting oleh Supabase dengan Row Level Security (RLS). Hanya Anda yang bisa mengakses data Anda.

Keamanan

  • Autentikasi menggunakan Supabase Auth (email/password atau Google OAuth)
  • Semua koneksi menggunakan HTTPS
  • Row Level Security memastikan data hanya bisa diakses oleh pemiliknya
  • Kami tidak pernah melihat atau mengakses data pengguna
  • Kode sumber terbuka (open source) untuk audit keamanan

Kontrol Data Anda

  • Ekspor: Fitur ekspor data akan segera tersedia
  • Edit & Hapus: Semua data bisa diedit atau dihapus langsung dari aplikasi
  • Hapus Akun: Hapus akun kapan saja melalui pengaturan

Layanan Pihak Ketiga

Financier menggunakan layanan berikut:

  • Supabase — Penyedia backend (auth, database, realtime). Kebijakan privasi Supabase
  • Google — OAuth login (opsional). Kebijakan privasi Google

Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui dari waktu ke waktu. Perubahan akan diumumkan melalui halaman ini dan/atau melalui aplikasi.

Kontak

Jika ada pertanyaan tentang kebijakan privasi ini, silakan buka issue di GitHub repository.
